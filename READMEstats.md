# Neonatal Bacterial Infection Analysis Pipeline

This repository contains the data processing and statistical analysis pipeline for the retrospective study on predicting neonatal bacterial infections using Electronic Health Record (EHR) data. 

The code processes dynamic physiological data (vital signs) and static demographic data to generate a univariate statistical comparison (Table 1) between neonates with culture-proven bacterial infections and non-infected controls.

## 📌 Overview of the Analysis
The Python script (`neonatal_infection_analysis.py`) performs the following operations:
1. **Data Aggregation:** Summarizes irregular, longitudinal physiological measurements (Heart Rate, SpO₂, Respiration, Temperature) into patient-level summary statistics (Mean, Min, Max, Standard Deviation, Percentiles).
2. **Event Extraction:** Quantifies the frequency and duration of specific clinical events (desaturations and bradycardias).
3. **Statistical Testing:** Compares the infected and non-infected cohorts using:
   - **Mann-Whitney U Test** for continuous non-parametric variables.
   - **Fisher's Exact / Chi-Square Test** for categorical variables.
4. **Effect Sizes:** Calculates Rank-Biserial Correlation ($r$) and Cramer's V ($V$) to quantify the magnitude of differences.
